Los estilos basados en propiedades aportan un camino simple para estilizar los componentes, disminuyendo el tiempo de desarrollo e incrementando la comprensión y mantenibilidad. 

## Referencia

A continuación la lista de propiedas soportadas y sus valores esperados.

### Background Color

```jsx
// bg="coolGray-500" hace referencia al valor `Theme.colors["coolGray-500"]`
<Box bg="coolGray-500">Box with background</Box>
```

| Prop                    | Propiedad Stylesheet | Tipo            |
| ----------------------- | -------------------- | --------------- |
| `bg`, `backgroundColor` | `backgroundColor`    | Theme['colors'] |

### Color

```jsx
// color="text" hace referencia al valor `Theme.colors["text"]`
<Text color="text">Text with color</Text>
```

| Prop    | Propiedad Stylesheet | Tipo            |
| ------- | -------------------- | --------------- |
| `color` | `color`              | Theme['colors'] |

| `ml`, `marginLeft`        | `marginLeft`         | Theme['spacing'] |
| `mx`, `marginHorizontal`  | `marginHorizontal`   | Theme['spacing'] |
| `my`, `marginVertical`    | `marginVertical`     | Theme['spacing'] |
| `ms`, `marginStart`       | `marginStart`        | Theme['spacing'] |
| `me`, `marginEnd`         | `marginEnd`          | Theme['spacing'] |
| `p`, `padding`            | `padding`            | Theme['spacing'] |
| `pt`, `paddingTop`        | `paddingTop`         | Theme['spacing'] |
| `pr`, `paddingRight`      | `paddingRight`       | Theme['spacing'] |
| `pb`, `paddingBottom`     | `paddingBottom`      | Theme['spacing'] |

### Layout

```jsx
<Box width="100%" height={200}>
  0
</Box>

<Box width="100%" height={50} flex={1} flexDirection="row" justifyContent="space-between">

    1
  </Box>
  <Box width={50} height={30} maxHeight={50}>
    2
  </Box>
  <Box width={30} maxWidth={40} height={50}>

  </Box>
</Box>
```

| `aspectRatio`       | `aspectRatio`        | <t>número</t>                                                                                        |
| `alignContent`      | `alignContent`       | <t>'flex-start' \| 'flex-end' \| 'center' \| 'stretch' \| 'space-between' \| 'space-around'</t>      |
| `alignItems`        | `alignItems`         | <t>'flex-start' \| 'flex-end' \| 'center' \| 'stretch' \| 'baseline' </t>                            |

### Text Shadow

```jsx
// textShadowColor="black" hace referencia al valor `Theme.colors["black"]`
<Text
  textShadowColor="black"
  textShadowOffset={{ width: 0, height: 2 }}
  textShadowRadius={4}
>
  Text with shadow
</Text>
```

| Prop               | Propiedad Stylesheet | Tipo                              |
| ------------------ | -------------------- | --------------------------------- |
| `textShadowColor`  | `textShadowColor`    | Theme['colors']                   |
| `textShadowOffset` | `textShadowOffset`   | { width: number, height: number } |
| `textShadowRadius` | `textShadowRadius`   | number                            |

### Shadow

```jsx
// shadowColor="coolGray-100" hace referencia al valor `theme.colors["coolGray-100"]`
<Box
  shadowOffset={{width: 20, height: 10}}
  shadowColor="coolGray-100"
  shadowRadius={13}
  shadowOpacity={0.7}
>
  Custom Shadow box
</Box>
```

| Prop            | Propiedad Stylesheet | Tipo                                     |
| --------------- | -------------------- | ---------------------------------------- |
| `elevation`     | `elevation`          | <t>número</t>                            |
| `shadowColor`   | `shadowColor`        | <t>Theme['colors']</t>                   |
| `shadowOffset`  | `shadowOffset`       | <t>{ width: número; height: número }</t> |
| `shadowOpacity` | `shadowOpacity`      | <t>número</t>                            |
| `shadowRadius`  | `shadowRadius`       | <t>número</t>                            |
