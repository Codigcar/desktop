# Introducción a Caral UI

Caral UI es un conjunto de guías y componentes cuyo propósito es lograr un desarrollo y diseño comprensible, eficiente, consistente y atractivo.

:::info
El nombre de Caral está basado en la **Ciudad Sagrada de Caral-Supe**, la que con 5000 años de antigüedad es considerada la ciudad más antigua de América y una de las más antiguas del mundo.
:::
