---
title: Password
---

El componente Password es una extensión de nuestro componente [Input](input), agregando un icono en la parte derecha que con un tap permite mostrar/ocultar la contraseña.

## Uso

```jsx
<Form ref={ref} onSubmit={onSubmit}>
  <Input.Password name="password" autoCapitalize="none" returnKeyType="send" />
</Form>
```
