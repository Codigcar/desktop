title: RadioGroup

---

El componente RadioGroup permite agrupar múltiples Radio's y poder seleccionar solo uno.

## Uso

```jsx
<Radio.Group>
  <Radio value="red" />
  <Radio value="green" />
  <Radio value="blue" />
</Radio.Group>
```

## Props

### `defaultValue`

Valor del Radio seleccionado por defecto.

| Tipo   | Requerido |
| ------ | --------- |
| string | no        |

### `name`

Usado como `testID`.

| Tipo   | Requerido |
| ------ | --------- |
| string | no        |

### `onChangeValue`

Es un CallBack que se llama cuando se hace tap sobre el Radio, y este contiene el valor del Radio seleccionado.

| Tipo                              | Requerido |
| --------------------------------- | --------- |
| (value: string,undefined) => void | no        |

## Métodos

| Nombre                             | Descripción                                             |
| ---------------------------------- | ------------------------------------------------------- |
| reset()                            | Selecciona el radio button por defecto (`defaultValue`) |
| setValue(value: string, undefined) | Define el valor del radio button                        |
| value                              | Devuelve el valor del radio seleccionado                |
