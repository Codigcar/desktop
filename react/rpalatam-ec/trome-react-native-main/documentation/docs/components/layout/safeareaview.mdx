El componente SafeAreaView es una abstracción del componente [SafeAreaView](https://github.com/th3rdwave/react-native-safe-area-context#safeareaview)
de la librería [react-native-safe-area-context](https://github.com/th3rdwave/react-native-safe-area-context),
agregando el beneficio de gestionar los estilos como propiedad. 

## Uso

```jsx
<SafeAreaView edges={['right', 'bottom', 'left']} mode="margin" height={200}>
  This is a safe area view
</SafeAreaView>
```
